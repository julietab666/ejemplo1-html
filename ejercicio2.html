¿Qué es HTML?

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos, como textos, imágenes, videos y enlaces. Utiliza un sistema de etiquetas (tags) para organizar elementos y crear la estructura básica que los navegadores interpretan para mostrar el sitio.

Historia de HTML

El HTML fue creado por Tim Berners-Lee en 1989-1991 en el CERN para facilitar el intercambio de documentos científicos. Evolucionó de ser un lenguaje de marcado básico para hipertexto a convertirse en el estándar de la web, pasando por versiones clave como HTML 2.0 (1995), HTML 4.0 (1997) y el revolucionario HTML5 (2014).

HTML5

HTML5 es la quinta y más moderna revisión del lenguaje de marcado HTML, estándar para estructurar y presentar contenido en la web. Introduce etiquetas semánticas (header, article), soporte nativo para multimedia (audio/video) sin plugins, gráficos 2D/3D (canvas), y APIs mejoradas para mejorar la accesibilidad, SEO y funcionalidad en navegadores modernos.
